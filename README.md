## Demo Register Avatar Profile User

First, run the development client:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Second, run the development server:

```bash
npm run start
# or
yarn start
```
